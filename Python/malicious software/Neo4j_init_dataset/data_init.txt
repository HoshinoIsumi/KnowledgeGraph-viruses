病毒特征
Satria.A 是一种隐形、驻留在内存的主引导记录 (MBR)/引导扇区感染病毒。感染后，Satria.A 成为内存驻留者，成为终止并保留驻留者 (TSR)。中断 13 被钩住。一旦驻留在内存中，Satria.A 就会在访问软盘时感染它们。

症状
Satria.A 包含以下测试字符串：《我亲爱的生日》《萨蒂亚》该病毒于 7 月 4 日激活。在这一天，它会显示“我爱你”的方块图形，用心表达爱。

感染方法
用MBR/引导扇区感染程序感染计算机的唯一方法是尝试从受感染的软盘引导。软盘的引导扇区具有确定软盘是否可引导的代码，并显示“非系统磁盘或磁盘错误”消息。正是该代码隐藏了感染。当出现非系统盘错误消息时，感染已经发生。病毒一旦执行，就会感染硬盘的MBR，并可能驻留在内存中。随后每次启动时，病毒都会被加载到内存中，并尝试感染机器访问的软盘。

删除说明
要删除病毒，请按照下列步骤操作：- 如果您使用 McAfee 紧急磁盘，请在出现 Windows 95 消息时按 F8，然后选择“逐步配置”。除了处理 autoexec.bat 文件之外，对所有事情都说“是”。- 在 a: 处，键入“SCAN C: /BOOT /CLEAN /NOMEM”
Windows NT/2000：
关闭 PC 并关闭电源。获取或创建无病毒启动盘和扫描盘。启动后，在A:\提示符下，执行以下命令：“SCAN C:/boot/clean”病毒被清除后，从计算机中取出所有软盘并从硬盘驱动器重新启动。这还将清理 NTFS 主引导记录并允许 Windows NT 从硬盘驱动器成功重新引导。VirusScan for DOS 将无法读取 NTFS 分区的其余部分。启动 Windows 后，执行 VirusScan 或 NetShield 来检测并清除可能存在的 Windows NT 文件感染。
